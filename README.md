# Pub-Sub-Basics-with-ZeroMQ


Este projeto é uma aplicação de chat simples implementada utilizando a biblioteca ZeroMQ. 
Ele permite o envio de mensagens tanto para usuários individuais quanto para tópicos, onde várias pessoas podem receber as mensagens direcionadas a um tópico específico.

Funcionalidades:

Envio de mensagens individuais para usuários específicos usando RPC.
Envio de mensagens para tópicos utilizando o padrão pub-sub.
Recebimento de mensagens enviadas para tópicos assinados.
